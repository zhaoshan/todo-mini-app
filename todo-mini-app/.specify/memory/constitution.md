# SimpleTodo MiniProgram Constitution

**项目名称**: SimpleTodo MiniProgram  
**项目类型**: 微信小程序 - 个人待办事项管理  
**技术栈**: 微信小程序原生框架 (WXML/WXSS/JavaScript)  
**Version**: 1.1.0  
**Ratified**: 2026-03-11  
**Last Amended**: 2026-03-11 (根据飞书文档更新)

---

## 🎯 核心目标

构建一个**极简风格的个人待办事项微信小程序**，仅包含"提醒"和"我的"两个页签，满足基础创建与展示功能，在微信开发者工具中可正常运行。

---

## 📋 开发原则

### 一、遵循官方规范 (OFFICIAL-STANDARD-FIRST)

- ✅ 严格遵循微信小程序官方开发规范
- ✅ 采用标准 WXML/WXSS/JS/JSON 项目结构
- ✅ 使用微信原生 API（`wx.getStorage`、`wx.setStorage`、`wx.getUserProfile` 等）
- ✅ 遵循小程序审核规范，避免违规内容

### 二、零外部依赖 (ZERO-EXTERNAL-DEPENDENCIES)

- ✅ 不依赖外部 npm 包
- ✅ 不使用云开发
- ✅ 纯本地存储方案
- ✅ 所有代码自主编写，可控可审计

### 三、极简主义 (MINIMALISM)

- ✅ 界面简洁、交互直观
- ✅ 适配主流手机屏幕
- ✅ 功能聚焦核心需求，避免过度设计
- ✅ 代码精简，易于维护

### 四、数据本地化 (LOCAL-FIRST)

- ✅ 仅支持本地存储（不联网、不登录后端）
- ✅ 使用 `wx.setStorageSync` / `wx.getStorageSync`
- ✅ 离线可用，无需网络连接
- ✅ 存储容量限制 10MB，需合理控制数据大小

### 五、测试驱动 (TEST-DRIVEN)

- ✅ 真机测试必须通过（iOS + Android）
- ✅ 微信开发者工具测试必须通过
- ✅ 关键功能手动验证清单
- ✅ 无控制台错误

## ⚠️ 功能边界

### 支持的功能

| 功能 | 说明 |
|------|------|
| 创建提醒 | 输入文本内容 + 选择时间 |
| 查看列表 | 按时间顺序显示所有提醒 |
| 切换状态 | 标记提醒为完成/未完成 |
| 删除提醒 | 长按删除单个提醒 |
| 编辑提醒 | 修改已有提醒的内容和时间 |
| 我的页面 | 静态信息展示（用户信息、使用说明） |

### 不支持的功能

| 功能 | 原因 |
|------|------|
| ❌ 重复提醒 | 超出极简范围 |
| ❌ 推送通知 | 需要后台支持 |
| ❌ 云端同步 | 违反本地化原则 |
| ❌ 用户登录 | 无需后端服务 |
| ❌ 分类/标签 | 增加复杂度 |
| ❌ 搜索功能 | 列表简短无需搜索 |
| ❌ 我的页面编辑 | 静态信息展示 |

## 🛠️ 技术约束

### 开发环境

| 项目 | 要求 |
|------|------|
| 微信开发者工具 | 最新版 |
| 基础库版本 | ≥ 2.19.0 |
| 调试基础库 | 2.19.0+ |
| 测试设备 | iOS + Android 真机 |

### 数据存储

| 类型 | API | 说明 |
|------|-----|------|
| 提醒列表 | `wx.setStorageSync('reminders', data)` | 同步写入，上限 10MB |
| 提醒读取 | `wx.getStorageSync('reminders')` | 同步读取 |
| 用户信息 | `wx.getUserProfile` | 获取微信用户信息（可选） |

### 代码规范

| 规范 | 要求 |
|------|------|
| 命名规范 | 驼峰命名（变量/函数），大驼峰（组件/页面） |
| 缩进 | 2 空格 |
| 单行长度 | ≤ 120 字符 |
| 注释 | 关键函数必须有注释 |
| 日志 | 生产环境移除 `console.log` |

### 性能要求

| 指标 | 目标 |
|------|------|
| 首屏加载 | ≤ 1 秒 |
| 列表渲染 | 100 条数据流畅滚动 |
| 存储操作 | 单次 ≤ 100ms |
| 页面切换 | ≤ 300ms |

## 📋 页面结构

### TabBar 配置

| 页签 | 路径 | 图标 | 说明 |
|------|------|------|------|
| 提醒 | `pages/reminders/index/index` | tab-reminder | 提醒列表页 |
| 我的 | `pages/mine/index/index` | tab-mine | 个人信息页 |

### 页面详情

#### 1. 提醒列表页 (pages/reminders/index/index)

**功能**:
- 显示所有提醒事项
- 按时间排序（未到期的在前）
- 点击切换完成状态
- 长按删除提醒
- 浮动按钮 "+" 创建新提醒
- 空状态显示"暂无提醒"

#### 2. 创建/编辑提醒页 (pages/reminders/create/create)

**功能**:
- 输入提醒内容（文本，≤200 字）
- 选择日期（不能选择过去日期）
- 选择时间
- 保存后返回列表
- 编辑模式回显原有数据

#### 3. 我的页面 (pages/mine/index/index)

**功能**:
- 显示用户头像和昵称（需授权）
- 未授权显示默认头像
- 使用说明文本
- 静态展示，不支持编辑
